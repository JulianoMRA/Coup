## Decisions Made

- `h-0 flex-1` on the GameLog scroll container requires the full parent flex chain (`flex flex-col flex-1 min-h-0`) to propagate height correctly — this is why the desktop panel wrapper was also updated
- ConnectionBadge fixed positioning removed; the GameBoard header already renders it inline next to the room title, so no layout break
- ExchangeSelector Badge → CharacterCard: the clickable div wrapper with conditional `ring-2 ring-primary` / `opacity-50` gives clearer visual feedback than the old Badge variant toggle
