Valid blocker rules (from 05-UI-SPEC.md):
| Action | Who Can Block | Card Claimed |
|--------|--------------|--------------|
| FOREIGN_AID | Any player | DUKE |
| STEAL | Target player only | CAPTAIN or AMBASSADOR |
| ASSASSINATE | Target player only | CONTESSA |
| TAX | Nobody | - |
| EXCHANGE | Nobody | - |

Challengeable actions: TAX, STEAL, ASSASSINATE, EXCHANGE (character claims). FOREIGN_AID is NOT challengeable (no character claim). INCOME and COUP are not challengeable.

        {/* Pending players */}
        {pendingPlayerNames && (
